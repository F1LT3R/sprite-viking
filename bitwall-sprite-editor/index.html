    Sprite-Viking-Editor 0.1-ish

    An HTML5 sprite cutting tool by F1lT3R

    Sprite-Viking was created to aid the development of The Mozilla Firefox4 Music Video. Sprite-Viking uses Charles Cliffe's Cubic-VR 3D engine, jQuery & jQuery UI to take care of business.

    There will probably be some documentation for this tool some day. Until that day, bug F1LT3R on irc://irc.mozilla.org/audio if you need help with Sprite-Viking.
